# LP 雙幣策略模擬器

互動式 AMM 流動性提供者 (LP) 雙幣策略模擬器，支援即時無常損失 (IL) 計算、手續費複利與挖礦獎勵貶值模擬。

![License](https://img.shields.io/badge/license-MIT-blue)

## 概覽

這個工具幫助你從兩種不同的角度，理解在恆定乘積 AMM（如 Uniswap v2 / Aerodrome）中提供流動性的**真實損益**：

| 角度 | 你原本持有 | 你的基準 |
|---|---|---|
| **情境 A — 存入 USDC** | 穩定幣 | HODL USDC |
| **情境 B — 存入 BTC** | BTC | HODL BTC |

兩種存入者最終獲得**完全相同的 LP 部位**（50/50 分配）。差異純粹在於**機會成本**。

## 核心公式

模擬器使用**恆定乘積** AMM 模型：

```
x · y = k

其中：
  x = 池中 BTC 數量
  y = 池中 USDC 數量
  k = 常數
```

**初始狀態**（投資金額 `I` 在價格 `p₀` 時 50/50 分配）：

```
y₀ = I / 2
x₀ = I / (2 · p₀)
k  = x₀ · y₀
```

**最終狀態**（價格為 `p₁` 時）：

```
x₁ = √(k / p₁)
y₁ = √(k · p₁)
LP 價值 = x₁ · p₁ + y₁ = 2 · √(k · p₁)
```

**無常損失** vs 50/50 HODL：

```
HODL 價值 = y₀ + x₀ · p₁
IL = (LP 價值 − HODL 價值) / HODL 價值
   = 2√r / (1 + r) − 1     其中 r = p₁ / p₀
```

**收益**（兩種來源，機制不同）：

```
手續費收益 = I × ((1 + feeAPR / 365)^天數 − 1)              // 每日複利
挖礦收益   = I × farmAPR × (天數 / 365) × 代幣留存率        // 線性累計，含代幣貶值
總收益     = 手續費收益 + 挖礦收益
```

**淨損益**：

```
情境 A (USDC)：總值 = LP 價值 + 總收益 − 投資金額
情境 B (BTC)： 總值 = (LP 價值 + 總收益) / p₁ − 初始 BTC 數量
淨年化 APY   = (總值 / 投資金額)^(365/天數) − 1
```

## 功能特色

- **中英雙語介面** — 一鍵切換語言
- 7 個互動滑桿：投資金額、持有天數、入場/出場價格、手續費 APR、挖礦 APR、獎勵代幣留存率
- **持有天數滑桿**（7–730 天）— 收益隨持有期間按比例計算
- **拆分收益模型** — 交易手續費 APR（每日複利）vs 挖礦獎勵 APR（線性累計）
- **獎勵代幣貶值模擬** — 可調整留存率，模擬挖礦代幣的價格下跌
- 即時雙視角損益（U本位 vs 幣本位）
- **收益拆解卡片** — 一覽手續費收益、挖礦收益、IL 與淨年化 APY
- AMM 底層機制展示池內資產組成變化
- 無常損失計算與視覺化指標
- 簡潔 Anthropic 風格的淺色主題

## 關鍵洞察

1. **LP = 賣出波動率。** 你放棄極端上漲（或避開極端下跌），換取穩定的手續費收入。
2. **BTC 存入者做 LP：** 當 BTC 大漲時表現不如 HODL（一半被轉成 USDC）。但若 BTC 暴跌，LP 能緩衝你的損失。
3. **USDC 存入者做 LP：** 相反 — 你獲得 BTC 上漲的曝險，但當 BTC 下跌時會受損（一半被轉成 BTC）。
4. **核心決策：** `APR > IL + 機會成本` 嗎？
5. **複利效果不可忽視。** 以 15% 手續費 APR 持有 2 年計算，每日複利比單利多賺約 5%。
6. **挖礦代幣風險是真實的。** 200% APR 毫無意義，如果獎勵代幣在你賣出時已跌了 80%。

## 已知簡化

- 價格從入場到出場瞬間跳變（無路徑模擬）；IL 不受路徑影響，但手續費收入是路徑依賴的。
- 手續費 APR 基於初始投資金額複利，而非變動中的 LP 價值。
- 無池子份額稀釋 — 將用戶視為整個池子。
- 無滑點、Gas 費或進出場費用。
- 假設 USDC 完美錨定 $1。
- 僅模擬 Uniswap v2 全範圍 LP（非集中流動性）。
